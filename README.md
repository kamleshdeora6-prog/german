# German Study App

GitHub-ready static study app.

## Files
- `index.html`
- `app.js`
- `passages.json`
- `notes.json`
- `exercises.json`
- `flashcards.json`
- `conjugation.json`
- `sentences.json`
- `vocab.json`
- `grammar_reference.png`

## Deploy on GitHub Pages
1. Upload all files to your repo root.
2. In GitHub repo settings, open **Pages**.
3. Set source to **Deploy from branch** and choose `main` + `/root`.
4. Save and wait for the site to publish.

All progress is stored locally in the browser using `localStorage`.
